* プログラミング言語の設定

** Mermaid

*** mmdc

Mermaid のソースコードから画像等を作成するコマンドラインツール。

**** mmdc のインストール

#+begin_src 
npm install -g @mermaid-js/mermaid-cli
#+end_src

**** Rakefile のサンプル

次の Rakefile を用意しておくと、ビルドが楽になります。

#+begin_src ruby
  PLANTUML = "/opt/homebrew/bin/plantuml"

  task :default => :uml

  desc "UML 図を更新する"
  task :uml => %w[
    foo.png
  ]

  rule '.png' => '.mmdc' do |t|
    sh "mmdc -i #{t.source} -o #{t.name}"
  end
#+end_src

#+begin_src ruby
  PLANTUML = "/opt/homebrew/bin/plantuml"

#+end_src
